# Algorithm_Exercises
这是算法设计与分析的练习题，每章节2-3个，都带有结果分析。搭配的是华中科技大学的《算法设计与分析》一书。

主要包括：递归法的应用、快速排序及其改进型、查找第K个元素、贪心算法、Prim和Kruskal算法、多段图动态规划算法、二分检索树的构建、实现双连通分图的输出及其改造算法等。
